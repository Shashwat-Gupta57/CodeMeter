# Contributing to CodeMeter

First off, thank you for considering contributing to CodeMeter!

## Development Setup

1. Ensure you have **Java 21** installed.
2. Clone the repository: `git clone https://github.com/Shashwat-Gupta57/CodeMeter.git`
3. Build the project using Gradle: `./gradlew build`
4. Run the TUI locally: `./gradlew run`

## Submitting Pull Requests

1. **Fork** the repository and create your feature branch: `git checkout -b feature/my-new-feature`.
2. **Commit** your changes following standard conventional commits.
3. **Test** your changes locally by running `./gradlew test`.
4. **Push** your branch and submit a Pull Request to the `main` branch.

## Bug Reports & Feature Requests

Please use GitHub Issues to report bugs or request features. Be as detailed as possible and provide logs or minimal reproducible examples if applicable.
