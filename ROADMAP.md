# CodeMeter Roadmap

CodeMeter is actively developed. Here is what we're planning for future releases:

## v2.6.0
- **CI/CD Integration**: First-class GitHub Action to measure PR size physically (e.g., "This PR adds 2 Burj Khalifas of code!").
- **HTML Export**: A self-contained, beautifully animated HTML report.
- **Plugin System**: Allow custom metrics (e.g., complexity converted to "coffee cups required").

## Future Explorations
- Distributed scanning for massive monorepos.
- AI-based summaries of code structure changes.
- Additional package manager support (Chocolatey, AUR, Snap).
