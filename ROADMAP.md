# CodeMeter Roadmap

CodeMeter is actively developed. Here is what we're planning for future releases:

## v1.1.0 (Q3)
- **CI/CD Integration Tools**: First-class GitHub Action to measure PR size physically (e.g. "This PR adds 2 Burj Khalifas of code!").
- **Custom Themes**: Add capability for users to define and load their own JSON themes for the TUI.
- **Deeper Historical Insights**: Graph view in TUI of code growth over the last 30 days.

## v1.2.0 (Q4)
- **Plugin System**: Allow custom metrics (e.g., complexity converted to "coffee cups required").
- **Interactive TUI Mouse Support**: Full mouse integration in Lanterna for clicking through the dashboard.
- **HTML Export**: A self-contained, beautifully animated HTML report.

## Future Explorations
- Distributed scanning for massive monorepos.
- AI-based summaries of code structure changes.
